# Alaya AI官网前端项目

## 项目概述
本项目是一个基于 Vue3 的 Alaya AI官网前端项目，主要面向用户展示企业动向，提供产品介绍，以及各种服务的展示。

## 技术栈
- 核心框架：Vue 3
- 状态管理：Pinia
- UI组件：Element Plus
- 路由管理：Vue Router
- 网络请求：Axios
- 构建工具：Vite
- 调试工具：Vue Devtools

## 项目结构
    src/
    |-- api/ 网络请求
    |-- assets/ 静态资源
    |-- components/ 组件
    |-- network/ 网络封装
    |-- pages/ 页面
    |-- App.vue 主组件
    |-- main.js 入口文件
    |-- style.css 样式文件
    |-- vite.config.ts 配置文件
    |-- package.json 依赖配置文件
    |-- README.md 项目说明文件
